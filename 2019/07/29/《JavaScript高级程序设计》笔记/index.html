第1章 JavaScript简介

Javascript是一种专门为网页交互而设计的语言，总体来说由三个部分组成：

  • ECMAScript，有ECMA-262定义，提供核心语言功能；
  • 文档对象模型（DOM），提供访问和操作网页内容的方法和接口；
  • 浏览器对象模型（BOM），提供与浏览器交互的方法与接口。

不同浏览器对Javascript的支持会有所差异。
